# Chapter Assets

This folder holds **rendered media** referenced by chapter docs.

- Layout: `docs/assets/ch-NN/<asset>.svg` (or `.png` / `.gif`).
- Optional per-chapter metadata: `docs/assets/ch-NN/media.json`.
- Generated assets only — generator scripts live in `scripts/media/`.

See `.github/instructions/chapter-media.instructions.md` for the full standard:
format priority (SVG → PNG → GIF), naming, alt text, and review rules.
